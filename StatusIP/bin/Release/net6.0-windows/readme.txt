Status IP - Cafe Martinez

Descripción: 
Aplicación para realizar un analisis de interconectividad entre los equipos de Cafe Martinez instalados en los locales.

Intruciones:
1- Ejecutar StatusIP 
2- Para configurar nombres y ip editar config.csv
3- Para modificar ip desde "Configuración" ingresar contraseña "hoguera"
4- Una vez configuradas las ip, tocar en inciar y esperar unos segundos a termine.

Direción del repositorio: https://github.com/FatuKING/statusip
